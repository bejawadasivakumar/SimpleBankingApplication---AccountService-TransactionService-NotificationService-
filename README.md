  # Services
- ## Account Service
     - API Endpoints:
       - POST /api/create - Create new account
       - GET /api/getDetails - Get all accounts
       - GET /api/AccountDetails/{accountnumber} - Get account details by accountnumber
       - GET /api/getBalance/{accountNumber} - Get account balance by accountNumber
       - GET /api/check/{accountNumber} - Check account is in DB or not
       - PUT /api/updateBalance/{accountNumber} - Update balance by accountNumber
       - DELETE /api/deleteAccount/{accountNumber} - Delete an account by accountNumber
 - ## Transaction Service
   - API Endpoints:
       - GET  /transactions/mini-statement/{accountNumber} - Get all transactions by accountNumber
       - POST /transactions/depositOrWithdraw - Get all accounts
 - ## Notification Service
   - Responsibilities:
       - Listen to RabbitMQ queues
       - Send email notifications(Create Account / WithDraw & Deposit)
       - Store notification records
